# 开始贡献

> 欢迎参与 LytJS 项目贡献！

---

## 如何开始

### 1. 了解项目

- 阅读 [架构设计](./architecture/8-layer-architecture.md) - 了解 8 层架构
- 查看 [开发规范](./development/guidelines.md) - 了解开发规范
- 学习 [开发流程](./development/workflow.md) - 了解完整的工作流程

### 2. 选择贡献类型

- 🐛 **修复 Bug** - 查看 [待办任务](./other/pending-tasks.md)
- ✨ **新功能** - 查看 [路线图](./roadmap/current.md)
- 📝 **文档** - 帮助完善文档
- 🧪 **测试** - 补充测试用例

### 3. 开发流程

1. Fork 并克隆仓库
2. 创建功能分支
3. 编写代码和测试
4. 确保所有测试通过
5. 提交 PR

---
