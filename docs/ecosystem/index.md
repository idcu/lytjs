# 生态系统

> LytJS 生态系统完整文档

---

## 📦 生态包分类

### 🧭 路由

路由相关包：

- **router** - 路由系统
- **router-fs** - 文件系统路由

### 🗄️ 状态管理

状态管理相关包：

- **store** - 状态管理（类似 Pinia）

### 🎨 UI 组件库

UI 组件库相关包：

- **ui** - UI 组件库
  - Vapor 模式支持

### 🌐 服务端渲染

服务端渲染相关包：

- **ssr** - 服务端渲染
  - 流式渲染
  - Island 架构
  - SSG 支持

### 🔧 开发工具

开发工具相关包：

- **devtools** - 开发工具
- **cli** - 命令行工具
- **test-utils** - 测试工具

### 🚀 构建工具

构建工具相关包：

- **bundler** - 构建工具集成
- **hmr** - 热模块替换
- **api** - API 工具
- **runtime-edge** - 边缘运行时

### 🔄 兼容性

兼容性相关包：

- **compat** - 兼容性包
