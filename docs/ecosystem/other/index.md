# 其他生态包

> 其他生态系统包文档

---

## 包列表

- **api** - API 工具
- **bundler** - 构建工具集成
- **hmr** - 热模块替换
- **runtime-edge** - 边缘运行时
- **compat** - 兼容性包
- **http-server** - HTTP 服务器
- **middleware** - 中间件核心
- **middleware-cors** - CORS 中间件
- **middleware-auth** - 认证中间件
- **middleware-rate-limit** - 限流中间件
- **metadata** - 元数据管理
- **ssg** - 静态站点生成
- **cache-isr** - ISR 缓存系统
- **html-renderer** - HTML 渲染器
