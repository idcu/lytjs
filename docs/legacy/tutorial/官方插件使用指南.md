## <a id="theme"></a> @lytjs/plugin-theme - 主题管理

### 简介

提供主题切换、管理和持久化功能。

### 安装与使用

```typescript
import { createApp } from '@lytjs/core';
import ThemePlugin from '@lytjs/plugin-theme';

const app = createApp(App);
app.use(ThemePlugin, {
  defaultTheme: 'light', // 默认主题
  themes: {
    light: { bg: '#fff', text: '#333' },
    dark: { bg: '#333', text: '#fff' },
    custom: { bg: '#f0f', text: '#0ff' }
  }
});
app.mount('#app');
```

### API

```typescript
// 在组件中使用
import { useTheme } from '@lytjs/plugin-theme';

const {
  currentTheme,    // 当前主题 signal
  isDark,          // 是否暗色模式 computed
  setTheme,        // 设置主题
  toggleTheme,     // 切换主题
  registerTheme    // 注册新主题
} = useTheme();

// 使用示例
toggleTheme(); // 切换明暗主题
setTheme('custom'); // 设置为自定义主题
```

## <a id="logger"></a> @lytjs/plugin-logger - 日志记录

### 简介

提供应用日志记录、过滤和持久化功能。

### 使用

```typescript
import LoggerPlugin from '@lytjs/plugin-logger';

app.use(LoggerPlugin, {
  level: 'debug', // 日志级别: 'debug'|'info'|'warn'|'error'
  persist: true,  // 是否持久化到本地存储
  maxLogs: 1000   // 最大日志数量
});
```

### API

```typescript
import { useLogger } from '@lytjs/plugin-logger';

const logger = useLogger();

logger.debug('调试信息');
logger.info('普通信息');
logger.warn('警告信息');
logger.error('错误信息');

// 获取历史日志
const logs = logger.getLogs();
// 清空日志
logger.clear();
```

## <a id="auth"></a> @lytjs/plugin-auth - 权限控制

### 简介

提供用户认证、权限检查和路由守卫功能。

### 使用

```typescript
import AuthPlugin from '@lytjs/plugin-auth';

app.use(AuthPlugin, {
  // 自定义权限验证函数
  checkPermission: (permission, user) => {
    return user.permissions.includes(permission);
  }
});
```

### API

```typescript
import { useAuth } from '@lytjs/plugin-auth';

const {
  user,            // 当前用户信息
  isAuthenticated, // 是否已认证
  login,           // 登录
  logout,          // 登出
  hasPermission,   // 检查权限
  hasRole          // 检查角色
} = useAuth();

// 登录
await login({ username: 'admin', password: '123456' });
// 检查权限
if (hasPermission('user:edit')) {
  // 有权限
}
```

## <a id="storage"></a> @lytjs/plugin-storage - 本地存储

### 简介

提供 localStorage/sessionStorage 的响应式封装。

### 使用

```typescript
import StoragePlugin from '@lytjs/plugin-storage';

app.use(StoragePlugin, {
  prefix: 'lyt_', // 键前缀
  serializer: {   // 自定义序列化
    stringify: JSON.stringify,
    parse: JSON.parse
  }
});
```

### API

```typescript
import { useStorage } from '@lytjs/plugin-storage';

const {
  get,
  set,
  remove,
  clear,
  useLocal,
  useSession
} = useStorage();

// 基本操作
set('key', 'value');
const value = get('key');
remove('key');

// 响应式存储
const count = useLocal('count', 0); // localStorage
const token = useSession('token', ''); // sessionStorage

count.value++; // 自动保存
```

## <a id="i18n"></a> @lytjs/plugin-i18n - 国际化

### 简介

提供多语言支持和翻译管理。

### 使用

```typescript
import I18nPlugin from '@lytjs/plugin-i18n';

app.use(I18nPlugin, {
  defaultLocale: 'zh-CN',
  fallbackLocale: 'en-US',
  messages: {
    'zh-CN': { hello: '你好', welcome: '欢迎，{name}' },
    'en-US': { hello: 'Hello', welcome: 'Welcome, {name}' }
  }
});
```

### API

```typescript
import { useI18n } from '@lytjs/plugin-i18n';

const {
  t,               // 翻译函数
  locale,          // 当前语言
  setLocale,       // 设置语言
  availableLocales // 可用语言列表
} = useI18n();

// 使用示例
console.log(t('hello')); // 你好
console.log(t('welcome', { name: 'LytJS' })); // 欢迎，LytJS
setLocale('en-US'); // 切换语言
```

## <a id="chart"></a> @lytjs/plugin-chart - 图表渲染

提供轻量级图表组件，基于 Canvas 实现。

```typescript
import ChartPlugin from '@lytjs/plugin-chart';
```

## <a id="form"></a> @lytjs/plugin-form - 表单管理与验证

### 简介

提供表单状态管理、验证和提交功能。

### 使用

```typescript
import FormPlugin from '@lytjs/plugin-form';

app.use(FormPlugin, {
  fields: {
    username: {
      initialValue: '',
      rules: [
        { type: 'required', message: '用户名必填' },
        { type: 'minLength', value: 3, message: '至少3个字符' }
      ]
    },
    email: {
      initialValue: '',
      rules: [
        { type: 'required' },
        { type: 'email' }
      ]
    }
  },
  validateOnSubmit: true,
  validateOnChange: false
});
```

### API

```typescript
import { useForm, createFormManager } from '@lytjs/plugin-form';

const form = useForm();
// 或创建独立表单实例
const myForm = createFormManager({ /* 配置 */ });

// API 方法
form.setValue('username', 'admin');
const value = form.getValue('username');
const values = form.getValues();

// 验证
const isValid = await form.validate();

// 提交
await form.submit(async (values) => {
  // 提交逻辑
  console.log(values);
});

// 表单状态
console.log(form.state.isValid);
console.log(form.state.isSubmitting);
```

### 内置验证规则

| 规则 | 说明 |
|------|------|
| required | 必填 |
| email | 邮箱 |
| phone | 手机号 |
| number | 数字 |
| min | 最小值 |
| max | 最大值 |
| minLength | 最小长度 |
| maxLength | 最大长度 |
| pattern | 正则 |
| custom | 自定义验证 |

## <a id="animation"></a> @lytjs/plugin-animation - 动画库

### 简介

提供声明式动画和过渡效果。

### 使用

```typescript
import AnimationPlugin from '@lytjs/plugin-animation';

app.use(AnimationPlugin);
```

### API

```typescript
import { useAnimation, animate } from '@lytjs/plugin-animation';

// 基本动画
await animate('.element', {
  from: { opacity: 0, transform: 'translateY(-20px)' },
  to: { opacity: 1, transform: 'translateY(0)' },
  duration: 300,
  easing: 'ease-out'
});

// 使用钩子
const { play, pause, stop } = useAnimation('#box', {
  keyframes: [
    { opacity: 0 },
    { opacity: 1 }
  ],
  duration: 500,
  iterations: Infinity
});

play();
pause();
```

### 预置动画

```typescript
import { fadeIn, slideInLeft, bounce } from '@lytjs/plugin-animation';

await fadeIn('.element');
await slideInLeft('.card');
await bounce('.button');
```

## <a id="data-fetch"></a> @lytjs/plugin-data-fetch - 数据获取与缓存

### 简介

提供数据请求、缓存和自动重试功能。

### 使用

```typescript
import DataFetchPlugin from '@lytjs/plugin-data-fetch';

app.use(DataFetchPlugin, {
  baseURL: 'https://api.example.com',
  cacheTime: 5 * 60 * 1000, // 5分钟
  staleTime: 60 * 1000      // 1分钟
});
```

### API

```typescript
import { useFetch, useSuspense } from '@lytjs/plugin-data-fetch';

// 基本使用
const { data, error, loading, refetch } = useFetch('/users');

// 带参数
const { data: user } = useFetch('/users/:id', { id: 1 });

// POST 请求
const { execute } = useFetch('/login', {
  method: 'POST',
  manual: true // 手动触发
});
await execute({ username: 'admin', password: '123' });

// Suspense 模式
const UserProfile = () => {
  const user = useSuspense('/users/1');
  return h('div', user.name);
};
```

## <a id="testing"></a> @lytjs/plugin-testing - 测试工具

### 简介

提供组件测试和应用测试的辅助工具。

### 使用

```typescript
import TestingPlugin from '@lytjs/plugin-testing';

app.use(TestingPlugin, {
  mockAPIs: true,
  snapshotTesting: true
});
```

### API

```typescript
import {
  mount,
  render,
  fireEvent,
  waitFor,
  createMock,
  getByTestId
} from '@lytjs/plugin-testing';

// 测试组件
const wrapper = mount(MyComponent, { props: { count: 0 } });

// 触发事件
await fireEvent.click(wrapper.getByRole('button'));

// 断言
expect(wrapper.text()).toContain('1');

// 模拟 API
createMock('/api/users', { method: 'GET', return: { data: [] } });
```

## 📚 插件组合使用示例

### 组合使用主题、国际化和存储

```typescript
import { createApp } from '@lytjs/core';
import ThemePlugin from '@lytjs/plugin-theme';
import I18nPlugin from '@lytjs/plugin-i18n';
import StoragePlugin from '@lytjs/plugin-storage';

const app = createApp(App);

// 注意插件顺序，StoragePlugin 先安装用于持久化其他插件状态
app.use(StoragePlugin);
app.use(ThemePlugin);
app.use(I18nPlugin);

app.mount('#app');
```

## 🎯 最佳实践

1. **插件顺序**：注意插件间的依赖关系，如 StoragePlugin 应在需要持久化的插件之前安装
2. **类型安全**：所有插件都提供完整的 TypeScript 类型支持
3. **性能考虑**：合理使用缓存，避免不必要的重新渲染
4. **错误处理**：始终处理异步操作中的错误情况
