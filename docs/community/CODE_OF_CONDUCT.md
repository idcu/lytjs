# LytJS 社区行为准则

## 我们的承诺

作为 LytJS 社区的成员、贡献者和领导者，我们承诺为每个人提供一个友好、安全和包容的参与环境，无论年龄、体型、残疾、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性别认同和表达。

## 我们的标准

### 鼓励的行为

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 与社区其他成员表达同理心

### 不可接受的行为

- 仇恨言论和歧视
- 人身攻击和侮辱
- 公开或私下骚扰
- 未经允许发布他人私人信息
- 其他不道德或不专业的行为

## 执法责任

社区领导者有责任澄清和执行本准则的可接受行为，并应采取适当和公平的纠正措施来应对任何他们认为不适当、威胁、冒犯或有害的行为。

## 执法指南

社区领导者将遵循以下原则来判定行为后果：

### 1. 纠正

**行为影响**: 社区成员超越界限
**后果**: 社区领导者的私下书面警告，说明如何改进

### 2. 警告

**行为影响**: 单次事件或一系列不当行为
**后果**: 公开警告，附带行为后果说明

### 3. 临时禁止

**行为影响**: 严重违反社区标准
**后果**: 临时禁止参与社区活动和交流

### 4. 永久禁止

**行为影响**: 持续不当行为或个人攻击模式
**后果**: 永久禁止参与社区

## 适用范围

本行为准则适用于所有社区空间，包括：
- GitHub 仓库和 Issue
- 官方论坛和聊天群
- 线下聚会和会议
- 其他官方活动
