# Common 系列包

> LytJS 公共工具包，零外部依赖，可独立使用

---

## 包列表

### 常量和类型

- **common/constants** - 通用常量
- **shared-types** - 共享类型定义

### 类型和断言

- **common/is** - 类型判断工具
- **common/assertions** - 类型断言

### 数据操作

- **common/object** - 对象和数组工具
- **common/string** - 字符串工具
- **common/algorithm** - 算法工具
- **common/vnode** - VNode 工具

### 异步和时序

- **common/timing** - 定时和 Promise 工具
- **common/scheduler** - 调度器
- **common/render-queue** - 渲染队列
- **common/raf** - RAF 工具

### DOM 和 Web

- **common/dom** - DOM 工具
- **common/dom-helpers** - DOM 辅助工具
- **common/web** - Web 平台工具
- **common/a11y** - 无障碍工具
- **common/keyboard** - 键盘工具

### 缓存和存储

- **common/cache** - 缓存工具
- **common/node-cache** - Node 缓存工具
- **common/storage** - 存储工具

### 错误和日志

- **common/error** - 错误处理工具
- **common/warn** - 警告和日志工具

### 其他工具

- **common/env** - 环境工具
- **common/events** - 事件工具
- **common/http** - HTTP 工具
- **common/path** - 路径工具
- **common/performance** - 性能工具
- **common/query** - 查询工具
- **common/security** - 安全工具
- **common/validate** - 验证工具
- **common/memory** - 内存工具
