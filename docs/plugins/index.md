# 插件生态

## 📦 插件分类

### 📊 数据管理

数据管理相关插件：

- **plugin-data** - 增强版数据获取插件
- **plugin-data-fetch** - 数据获取基础插件
- **plugin-form** - 表单管理插件

### 🛡️ 认证和安全

认证和安全相关插件：

- **plugin-auth** - 认证插件

### 🌐 国际化

国际化相关插件：

- **plugin-i18n** - 国际化插件

### 🎨 UI 和主题

UI 和主题相关插件：

- **plugin-chart** - 图表插件
- **plugin-theme** - 主题插件

### 🔧 开发工具

开发工具相关插件：

- **plugin-logger** - 日志插件
- **plugin-vite** - Vite 集成插件
