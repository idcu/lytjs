## 🔧 运行示例

```bash
# 克隆项目
git clone https://gitee.com/lytjs/lytjs.git

# 进入示例目录
cd examples

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后在浏览器中打开相应的 HTML 文件即可。

## 💡 核心学习要点

### 响应式系统
- `signal` - 基础响应式状态
- `computed` - 计算属性
- `effect` - 副作用监听
- `watch` - 观察数据变化

### 组件系统
- 组件定义和注册
- Props 传递
- 事件发射和监听
- 插槽使用

### 生态系统
- Router - 路由导航
- Store - 状态管理
- UI - 组件库
- Plugins - 插件系统
