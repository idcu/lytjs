## 🏗️ 项目结构快速参考

```
lytjs/
├── docs/                          # 文档（你在这里）
│   ├── guide/                    # 用户指南
│   ├── api/                      # API 参考
│   ├── developer/                # 开发者文档
│   └── examples/                 # 示例代码
├── packages/                      # 核心包
│   ├── reactivity/               # 响应式系统
│   ├── compiler/                 # 编译器
│   ├── vdom/                     # 虚拟 DOM
│   ├── renderer/                 # 渲染器
│   ├── component/                # 组件系统
│   ├── core/                     # 核心入口
│   ├── router/                   # 路由
│   ├── store/                    # 状态管理
│   ├── cli/                      # 命令行工具
│   ├── devtools/                 # 开发者工具
│   ├── components/               # UI 组件库
│   ├── lytx/                     # 元框架
│   ├── lytjs/                    # 聚合包
│   └── vscode-extension/         # VSCode 扩展
├── examples/                      # 示例项目
├── benchmarks/                    # 性能基准
├── tests/                         # 测试文件
└── scripts/                       # 构建脚本
```
