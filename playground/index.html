Lyt.js 快速入门

安装

npm install @lytjs/core

基础使用

import { createApp, defineComponent, ref } from '@lytjs/core';

const App = defineComponent({
  setup() {
    const count = ref(0);
    return { count };
  },
  template: '<div>Count: {{ count }}</div>'
});

createApp(App).mount('#app');

核心概念

  • ref - 响应式引用，用于基本数据类型
  • reactive - 响应式对象，用于复杂数据类型
  • computed - 计算属性，根据依赖自动更新
  • watch - 侦听器，监听数据变化

模板语法

Lyt.js 使用无 v- 前缀的模板语法：

<div if="show">显示条件内容</div>
<ul>
  <li each="item in items" :key="item.id">{{ item.text }}</li>
</ul>
<button @click="handleClick">点击</button>
