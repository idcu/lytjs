# Lyt.js Playground

Lyt.js 在线体验环境 - 快速尝试和学习 Lyt.js。

## 快速开始

### 本地运行

```bash
cd playground
npm install
npm run dev
```

访问 http://localhost:3000 即可使用 Playground。

### 在 StackBlitz 上运行

点击下方按钮在 StackBlitz 上直接运行：

[![Open in StackBlitz](https://developer.stackblitz.com/img/open_in_stackblitz.svg)](https://stackblitz.com/github/lytjs/lytjs/tree/main/playground)

## 功能特性

- 📝 在线编辑器，支持编写 Lyt.js 代码
- 🎯 实时预览渲染结果
- 📚 预定义示例代码
- 📖 内置文档
- 🚀 零配置运行

## 使用说明

1. 在编辑器中编写 Lyt.js 代码
2. 点击「运行代码」按钮查看效果
3. 切换「示例」标签页查看预设示例
4. 切换「文档」标签页查看文档

## 示例列表

- **计数器** - 基础响应式和计算属性示例
- **Todo 列表** - 完整的 Todo 应用
- **表单** - 表单验证和数据绑定
- **组件** - 组件定义、props 和事件传递
