feat: add @lytjs/plugin-validation

新增 @lytjs/plugin-validation 验证插件，提供以下功能：

- 12+ 内置验证规则：required, email, phone, number, min, max, minLength, maxLength, length, pattern, url, uuid, date
- 类型安全的验证 API
- 支持同步和异步验证
- 支持自定义验证规则和错误消息
- 支持 stopOnFirstError 选项
- 与 @lytjs/plugin-form 深度集成
- 完整的测试覆盖和文档
