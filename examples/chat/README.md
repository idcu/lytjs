# 实时聊天

一个使用 Lyt.js 构建的实时聊天应用，展示响应式数据更新和模拟实时通信。

## 功能特性

- **联系人列表**：6 位模拟联系人，显示在线状态
- **实时聊天**：发送消息并接收模拟回复
- **正在输入提示**：对方回复前显示"正在输入"动画
- **消息搜索**：搜索联系人快速定位
- **自动滚动**：新消息自动滚动到底部
- **响应式设计**：适配桌面和移动端

## 技术栈

- **Lyt.js** `@lytjs/core` v5.0.0（CDN 加载）
- 纯 HTML + CSS + JavaScript，无需构建工具
- 模拟 WebSocket 实时通信（setTimeout）

## 展示的框架特性

| 特性 | 说明 |
|------|------|
| `createApp` | 创建应用实例 |
| `ref` | 响应式引用（消息列表、输入内容、在线状态） |
| `computed` | 计算属性（联系人过滤、当前消息列表） |
| `watchEffect` | 副作用（消息变化时自动滚动） |
| `watch` | 侦听器 |
| `nextTick` | DOM 更新后滚动到底部 |
| `template` | 模板语法（v-for, v-if, v-model, :class, :style, @click, @keydown） |
| `setup()` | 组合式 API，封装完整聊天逻辑 |

## 运行方式

1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   npx serve .
   ```
