# Dashboard - 数据仪表盘

一个使用 Lyt.js 构建的数据仪表盘应用，展示路由、状态管理和响应式布局。

## 功能特性

- **侧边栏导航**：支持折叠/展开，多页面切换
- **统计卡片**：展示关键业务指标（收入、订单、用户、转化率）
- **数据表格**：订单列表，支持搜索过滤
- **主题切换**：暗色/亮色主题，CSS 变量驱动
- **响应式布局**：适配桌面和移动端

## 技术栈

- **Lyt.js** `@lytjs/core` v5.0.1（CDN 加载）
- 纯 HTML + CSS + JavaScript，无需构建工具
- CSS 变量实现主题系统

## 展示的框架特性

| 特性 | 说明 |
|------|------|
| `createApp` | 创建应用实例 |
| `ref` | 响应式引用（当前页面、搜索词、主题状态） |
| `computed` | 计算属性（页面标题、搜索过滤） |
| `watch` | 侦听器（主题切换时更新 DOM） |
| `template` | 模板语法（v-for, v-if, v-model, :class, :style, @click） |
| `setup()` | 组合式 API，封装完整业务逻辑 |

## 运行方式

1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   npx serve .
   ```
