# 电商首页

一个使用 Lyt.js 构建的电商首页，展示商品列表、搜索过滤和购物车功能。

## 功能特性

- **商品列表**：12 件模拟商品，网格布局展示
- **分类筛选**：数码、运动、服饰、生活四大分类
- **搜索过滤**：实时搜索商品名称和描述
- **价格排序**：默认 / 价格升序 / 价格降序 / 折扣优先
- **购物车**：侧边抽屉式购物车，支持增减数量和删除
- **响应式设计**：适配桌面、平板和手机

## 技术栈

- **Lyt.js** `@lytjs/core` v5.0.1（CDN 加载）
- 纯 HTML + CSS + JavaScript，无需构建工具

## 展示的框架特性

| 特性 | 说明 |
|------|------|
| `createApp` | 创建应用实例 |
| `ref` | 响应式引用（搜索词、分类、排序、购物车） |
| `computed` | 计算属性（过滤排序、购物车统计） |
| `template` | 模板语法（v-for, v-if, v-model, :class, @click） |
| `setup()` | 组合式 API，封装完整电商逻辑 |

## 运行方式

1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   npx serve .
   ```
