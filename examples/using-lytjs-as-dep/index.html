🚀 使用 LytJS 作为依赖

📖 项目介绍

这是一个完整的示例，展示如何将 LytJS 作为第三方依赖引入到你的项目中， 以及如何使用 AI 辅助开发流程。

LytJS 是一个现代的 JavaScript 响应式框架，支持双模式渲染： Vapor 模式（高性能 Signal 驱动）和 VDOM 模式（兼容性优先）。

查看 LytJS 仓库 返回示例列表

🛠️ 快速安装

在你的项目中安装 LytJS：

npm install @lytjs/core @lytjs/ui @lytjs/store @lytjs/router
