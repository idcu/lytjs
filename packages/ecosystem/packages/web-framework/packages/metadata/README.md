# @lytjs/metadata

> LytJS 元数据管理系统。

[![npm version](https://img.shields.io/npm/v/@lytjs/metadata.svg)](https://www.npmjs.com/package/@lytjs/metadata)
[![license](https://img.shields.io/npm/l/@lytjs/metadata.svg)](https://gitee.com/lytjs/lytjs/blob/main/LICENSE)

## 简介

`@lytjs/metadata` 是 LytJS 框架的元数据管理系统，支持 HTML meta 标签、OpenGraph、Twitter Cards 等。

### 核心特性

- **链式 API**：流畅的链式调用方式
- **OpenGraph 支持**：完整的 OpenGraph 元数据
- **Twitter Cards 支持**：支持 Twitter 卡片
- **SEO 优化**：自动处理 SEO 相关元数据
- **零依赖**：不引入任何外部依赖

## 安装

```bash
npm install @lytjs/metadata
```

或使用 pnpm：

```bash
pnpm add @lytjs/metadata
```
