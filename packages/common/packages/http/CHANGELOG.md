# @lytjs/common-http 更新日志

## [6.6.0] - 2026-05-21

### 🚀 功能增强

- 新增便捷方法：`get`, `post`, `put`, `patch`, `del`
- 新增直接返回数据的 JSON 方法：`getJson`, `postJson`, `putJson`, `patchJson`, `deleteJson`, `requestJson`
- 集成 `@lytjs/common-query`，支持数组查询参数
- 新增完整的测试用例

### 📦 依赖

- `@lytjs/common-query` 更新至 `^6.6.0`

## [6.5.0] - 2026-05-19
